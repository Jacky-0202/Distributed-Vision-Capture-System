### 🛠️ 實戰解題：$F(s) = \frac{3}{s(s^2+2s+5)}$

#### 第一步：分析分母

1. **實根**：$s = 0$。
2. **複數根**：$s^2+2s+5$。
    
    - 判別式 $D = 2^2 - 4(1)(5) = -16 < 0$，這代表有**衰減震盪**。
    - **配方**：$s^2+2s+5 = (s+1)^2 + 4 = (s+1)^2 + 2^2$。

#### 第二步：列出拆解模板

根據分母結構，我們設定：

$$\frac{3}{s(s^2+2s+5)} = \frac{A}{s} + \frac{Bs + C}{s^2+2s+5}$$

> **注意**：二次項分母上面的分子必須是一次式 $Bs+C$。

#### 第三步：求解係數

1. **求 A (覆蓋法)**：
    
    遮住左邊的 $s$，代入 $s=0$：
    
    $$A = \frac{3}{0^2 + 2(0) + 5} = \frac{3}{5} = \mathbf{0.6}$$
    
2. **求 B 和 C (比較係數法)**：
    
    將兩邊通分合併分子：
    
    $$3 = 0.6(s^2+2s+5) + (Bs+C)s$$
    
    $$3 = (0.6+B)s^2 + (1.2+C)s + 3$$
    
    - 看 **$s^2$** 項：$0.6 + B = 0 \implies \mathbf{B = -0.6}$
    - 看 **$s$** 項：$1.2 + C = 0 \implies \mathbf{C = -1.2}$

**目前結果：**

$$F(s) = \frac{0.6}{s} + \frac{-0.6s - 1.2}{(s+1)^2 + 2^2}$$

#### 第四步：分子整形（為了配對 $\sin$ 和 $\cos$）

這是最關鍵的一步！我們要讓分子的 $s$ 變成和分母一樣的 **$(s+1)$**。

- 分子是 $-0.6s - 1.2$。
- 我們強行寫出 $(s+1)$：
    
    $$-0.6(s+1) = -0.6s - 0.6$$
    
- 為了補回原本的 $-1.2$，後面還要再減掉 $0.6$：
    $$-0.6s - 1.2 = \mathbf{-0.6(s+1) - 0.6}$$
    

#### 第五步：最終組合零件

現在把式子拆乾淨：

$$F(s) = \underbrace{\frac{0.6}{s}}_{\text{常數}} \underbrace{- 0.6 \frac{(s+1)}{(s+1)^2 + 2^2}}_{\text{位移後的 } \cos} \underbrace{- 0.6 \frac{1}{(s+1)^2 + 2^2}}_{\text{準備變 } \sin}$$

為了讓最後一項符合 $\frac{k}{s^2+k^2}$，我們知道 $k=2$，所以分子要湊出 $2$：

$$- 0.6 \frac{1}{(s+1)^2 + 2^2} = - 0.3 \frac{2}{(s+1)^2 + 2^2}$$

#### 第六步：反拉氏轉換（回到 $t$ 領域）

模板：
- **餘弦模板 (Cosine)**：$\mathcal{L}\{\cos(kt)\} = \frac{s}{s^2+k^2}$
- **正弦模板 (Sine)**：$\mathcal{L}\{\sin(kt)\} = \frac{k}{s^2+k^2}$
- **位移法則 (Shifting)**：如果 $s$ 被換成了 $(s-a)$，代表在 $t$ 世界要乘上 $e^{at}$

解答：
1. $0.6/s \rightarrow \mathbf{0.6}$
2. $-0.6 \frac{(s+1)}{(s+1)^2 + 2^2} \rightarrow \mathbf{-0.6 e^{-t} \cos(2t)}$
3. $-0.3 \frac{2}{(s+1)^2 + 2^2} \rightarrow \mathbf{-0.3 e^{-t} \sin(2t)}$

---
### 🏁 最終答案：

$$f(t) = 0.6 - 0.6 e^{-t} \cos(2t) - 0.3 e^{-t} \sin(2t)$$
