## 1. 核心思維：為什麼要拆？

拉氏轉換表裡面只有簡單的形狀。當你看到：

$$F(s) = \frac{1}{(s+1)(s+2)}$$

字典裡查不到這整坨東西。但如果你能把它拆成：

$$\frac{A}{s+1} + \frac{B}{s+2}$$

這兩塊我們閉著眼睛都能還原成 $e^{-t}$ 和 $e^{-2t}$。
**部分分式就是這場「逆向加法」的遊戲。**

---
## 2. 第一招：簡單實根 (Heaviside 覆蓋法) —— 最強秒殺技

這是處理分母是不同一次項（如 $(s-a)(s-b)$）時最快的方法。

**題目：** $F(s) = \frac{s+7}{(s+1)(s-2)}$

**步驟：**

1. **設定目標**：$\frac{s+7}{(s+1)(s-2)} = \frac{A}{s+1} + \frac{B}{s-2}$
2. **求 A (遮住 s+1)**：
    
    - 在左邊式子遮住 $(s+1)$，剩下 $\frac{s+7}{s-2}$。
    - 把 $s = -1$（讓被遮住的項變 0 的數）代入剩下部分：
    - $A = \frac{-1+7}{-1-2} = \frac{6}{-3} = -2$

3. **求 B (遮住 s-2)**：

    - 在左邊式子遮住 $(s-2)$，剩下 $\frac{s+7}{s+1}$。
    - 把 $s = 2$ 代入：
    - $B = \frac{2+7}{2+1} = \frac{9}{3} = 3$

**結果：** $F(s) = \frac{-2}{s+1} + \frac{3}{s-2} \xrightarrow{\mathcal{L}^{-1}} -2e^{-t} + 3e^{2t}$

---
## 3. 第二招：重複實根 (Repeated Roots)

當分母出現次方，如 $(s+a)^n$ 時，拆法要稍微變通，必須「逐層降級」。

**範例：** $\frac{1}{s(s+1)^2} = \frac{A}{s} + \frac{B}{s+1} + \frac{C}{(s+1)^2}$

- **注意**：$(s+1)^2$ 必須拆出一次方和二次方兩項。
- **解法**：最外層的 $A$ 和最高次的 $C$ 依然可以用覆蓋法。中間的 $B$ 則通常用代入法（隨便代一個簡單的 $s$，如 $s=1$）或兩邊通分比較係數。

---
## 4. 第三招：不可約二次項 (Irreducible Quadratic)

當分母出現像 $s^2+4$ 這種拆不開的一次項時：

**範例：** $\frac{s+1}{s(s^2+4)} = \frac{A}{s} + \frac{Bs+C}{s^2+4}$

- **關鍵**：分母是二次方時，分子必須假設成一次方 ($Bs+C$)。
- 這類題目通常會導向現實世界的 **$\sin$ 或 $\cos$**。

## 💡 你的「邏輯證明」：為什麼覆蓋法會成功？

你一定會懷疑：憑什麼「遮住」就能算出係數？

**讓我們看 $A$ 的求法：**

$$\frac{s+7}{(s+1)(s-2)} = \frac{A}{s+1} + \frac{B}{s-2}$$

兩邊同時乘上 $(s+1)$：

$$\frac{s+7}{s-2} = A + \frac{B(s+1)}{s-2}$$

現在，如果我們讓 **$s = -1$**：
右邊的第二項 $\frac{B(-1+1)}{-1-2}$ 會直接變成 **0**！
剩下的式子就是：$\frac{-1+7}{-1-2} = A$。

**這就是覆蓋法的原理：透過乘法和取極限，把不相關的項通通殺掉，只留下你要的係數。**

## 🧠 筆記總結：反拉氏解題流程

1. **觀察分母**：能不能分解？
2. **分類拆解**：
    
    - 一次方 $\to$ 分子放常數。
    - 二次方 $\to$ 分子放 $As+B$。
        
3. **執行覆蓋法**：快速決定常數。
4. **對照字典**：還原回 $t$ 世界。
