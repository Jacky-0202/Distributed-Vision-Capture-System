## 摘要

 **核心定義**：資訊從輸入層開始，經過每一層隱藏層的計算與轉換，最終抵達輸出層並產生預測結果的**單向傳遞**過程。
 
 **三個關鍵價值**：
 
 1. **預測基礎**：它是模型「推理 (Inference)」的核心動作。
 2. **層級運算**：將複雜的函數分解成多個簡單的矩陣運算組合。     
 3. **損失計算的前導**：必須先有前向傳播的結果，才能計算誤差（Loss）並進行後續學習。

---
## 🛠️ 數據流動的「三步演算法」

在每一層神經網路中，前向傳播都在重複這三個動作：

1. **加權與偏移 (Linear Combination)**：

    計算該層的原始得分 $z$。
    
    $$z^{(l)} = W^{(l)} a^{(l-1)} + b^{(l)}$$
    
2. **活化轉換 (Non-linear Activation)**：
    
    將得分傳入激勵函數 $\sigma$，得到活化後的輸出 $a$。
    
    $$a^{(l)} = \sigma(z^{(l)})$$
    
3. **向後傳遞 (Pass Forward)**：

    將這層的輸出 $a^{(l)}$ 作為下一層的輸入。

| **符號**      | **術語名稱** | **為什麼要這樣寫？**                |
| ----------- | -------- | --------------------------- |
| **$(l)$**   | 上標層數     | 因為網路有很多層，$(l)$ 就是這層的「門牌號碼」。 |
| **$z$**     | 加權總和     | 代表還沒經過激勵函數過濾的純數據。           |
| **$W$**     | 權重矩陣     | 決定前一層資訊的重要性。                |
| **$a$**     | 活化輸出     | $a$ 代表 **Activation**。      |
| **$(l-1)$** | 前一層上標    | 代表輸入這層的資料，其實是「上一層」吐出來的成果。   |
| **$b$**     | 偏置向量     | 我們最愛的那個「偏置」。                |
