### 第一階段：準備工作

1. **下載 Ubuntu ISO 鏡像檔**： 前往 [Ubuntu 官網](https://ubuntu.com/download/desktop) 下載最新的 LTS 版本（目前建議 24.04 ），LTS 版本較為穩定且支援期長。
2. **準備 USB 隨身碟**： 容量至少需要 **8GB**，請注意隨身碟內的資料會被清空。
3. **備份目標電腦資料**： 重新灌系統會抹除硬碟，請確保另一台電腦的重要資料已經備份。

---
### 第二階段：在目前的 Ubuntu 製作開機碟

Ubuntu 內建了一個非常直覺的工具，不需要額外安裝軟體：

1. 按下 `Super` 鍵（Windows 鍵），搜尋 **「啟動磁碟製作程式」(Startup Disk Creator)** 並開啟它。
2. 在「來源光碟映像檔 (.iso)」中，選擇你剛下載的 Ubuntu ISO 檔。
3. 在「要使用的磁碟」中，選擇你的 USB 隨身碟。
4. 點擊 **「製作啟動磁碟」** 並輸入密碼確認。

---
### 第三階段：開始安裝（另一台電腦）

1. **插上 USB 並重啟**：將製作好的隨身碟插到另一台電腦上。
2. **進入開機選單 (Boot Menu)**： 電腦重啟時，狂按快捷鍵（常見為 `F12`、`F11`、`F10` 或 `Esc`，視主機板品牌而定）。
3. **選擇 USB 裝置**：在開機選單中選擇你的 USB 隨身碟進入。
4. **進入安裝程式**：
    
    - 選擇 **"Try or Install Ubuntu"**。
    - 進入桌面後，點擊 **"Install Ubuntu"** 圖示。

5. **跟隨指引設定**：
    
    - **鍵盤佈局**：通常選 Chinese (Taiwan)。
    - **安裝類型**：如果是要全新安裝，選擇 **「抹除磁碟並安裝 Ubuntu」** (Erase disk and install Ubuntu)。
    - **時區與帳號**：設定你的使用者名稱與密碼。

---

### 第四階段：後續優化建議

安裝完成後，記得執行以下指令確保系統是最新的：

```
sudo apt update && sudo apt upgrade -y
```

如果你之後需要在這台新電腦上配置你習慣的開發環境（例如 Python、Docker 或是 Obsidian），這會是個好時機。
