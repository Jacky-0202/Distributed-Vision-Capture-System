## 1. 拆解規則：逐層降級

假設你的分母有 $(s+a)^n$，你的部分分式必須長這樣：

$$\frac{A_n}{(s+a)^n} + \frac{A_{n-1}}{(s+a)^{n-1}} + \dots + \frac{A_1}{s+a}$$

**例子：** 如果分母是 $(s+1)^2$，你就要拆成：

$$\frac{A}{(s+1)^2} + \frac{B}{s+1}$$

---
## 2. 實戰演練：覆蓋法＋代入法

我們拿這題來開刀：

$$F(s) = \frac{1}{s(s+1)^2}$$

#### 第一步：寫下模板

$$F(s) = \frac{A}{s} + \frac{B}{s+1} + \frac{C}{(s+1)^2}$$

#### 第二步：用「覆蓋法」抓出最容易的係數

覆蓋法只能抓到「最純」的那幾項（即沒有重複的，或是重複項中**最高次方**的那一個）：

1. **求 A (遮住 $s$，代入 $s=0$)**：
    
    $$A = \frac{1}{(0+1)^2} = 1$$
    
2. **求 C (遮住 $(s+1)^2$，代入 $s=-1$)**：
    
    _注意：這只能求出最高次方的 $C$，求不出 $B$！_
    
    $$C = \frac{1}{-1} = -1$$
    

#### 第三步：求「隱藏係數」$B$ (使用代入法)

現在我們知道 $A=1, C=-1$，式子變成了：

$$\frac{1}{s(s+1)^2} = \frac{1}{s} + \frac{B}{s+1} + \frac{-1}{(s+1)^2}$$

為了算出 $B$，我們隨便代入一個簡單的 $s$ 值（只要不讓分母變 0 即可）。

**令 $s = 1$**：

$$\frac{1}{1(2)^2} = \frac{1}{1} + \frac{B}{2} + \frac{-1}{2^2}$$

$$\frac{1}{4} = 1 + \frac{B}{2} - \frac{1}{4}$$

整理一下：$\frac{1}{2} = 1 + \frac{B}{2} \implies \frac{B}{2} = -\frac{1}{2} \implies \mathbf{B = -1}$。

## 3. 反拉氏轉換：見證「時間增長」

現在零件都齊了：

$$F(s) = \frac{1}{s} - \frac{1}{s+1} - \frac{1}{(s+1)^2}$$

我們一項一項轉回 $t$ 領域：

1. $\frac{1}{s} \xrightarrow{\mathcal{L}^{-1}} 1$
2. $\frac{1}{s+1} \xrightarrow{\mathcal{L}^{-1}} e^{-t}$
3. **$\frac{1}{(s+1)^2} \xrightarrow{\mathcal{L}^{-1}} t e^{-t}$** （這就是我們之前證明的 $\frac{1}{s^2}$ 位移後的結果！）


**最終答案：**

$$f(t) = 1 - e^{-t} - t e^{-t}$$
