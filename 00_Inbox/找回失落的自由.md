## 一、 引言：我們真的自由嗎？ 

哈利路亞～各位弟兄姊妹，平安。 今天想跟大家分享的主題是：「找回失落的自由」。 
大家聽到這個題目可能會想，怎麼會定這個主題呢？難道我們現在不自由嗎？

提到自由，很多人可能會想到匈牙利詩人裴多菲的一首名詩：
```
生命誠可貴，愛情價更高，若為自由故，兩者皆可拋
```

這句詩深刻地描繪出，在自由面前，連生命與愛情都可以放下。
人類的歷史，幾乎就是一部爭取自由的歷史。

而回到我們現在的生活，從台灣的法律來看，憲法確實保障了我們許多權利：
言論自由、信仰自由、居住跟遷徙的自由。
簡單來說，法律定義的自由就是：「只要不違法、不侵犯別人，我想做什麼就可以做什麼，沒有人可以隨便限制我。」

如果單從這個角度來看，我們確實活在一個極度自由的國度。我們想吃什麼、去哪裡玩、要選擇什麼樣的信仰，
都可以憑著神造人的時後所賜給我們的「自由意志」來決定 。

但是，法律保障了我們身體和行為的自由，我們的心靈，真的自由嗎？

翻開聖經
```
彼得後書 2:19
他們應許人得以自由，自己卻做敗壞的奴僕，因為人被誰制伏，就是誰的奴僕。
```

親愛的弟兄姊妹們，其實我們今天遇到的這種『假自由』，在兩千年前的教會就發生過了。

當時的使徒彼得，面對教會裡的一群『假師傅』。
這些人到處跟信徒開空頭支票說：「信耶穌了，我們自由啦！肉體想做什麼都可以，盡情去享受吧！」
這聽起來，是不是跟我們現在社會常聽到的「只要我喜歡，有什麼不可以」？

但彼得直接戳破了這個謊言。彼得的意思是，這些人嘴裡喊著自由的口號，
實際上他們根本控制不了自己的慾望。他們被貪婪制伏、被情慾制伏，說穿了，他們其實是慾望的奴隸！

這句話套用在我們現代人身上，簡直如出一轍。
當我們以為放縱自己去沉迷網路與社群、去抽菸酗酒是一種自由的時候，
其實我們已經被這些習慣給『制伏』了，不知不覺中，我們成了這些壞習慣的奴僕。

這就是我今天想要探討的核心。神明明給了我們尊貴的自由意志，為什麼我們最後會落入這種被制伏、不自由的狀態呢 ？
其實，這一切都要回到我們「人性的軟弱」來談。
接下來，我們就來看看，這種軟弱是如何讓我們濫用了自由意志，以及現代的文化跟風俗又是怎麼轄制我們的……

## 二、 人性的軟弱

#### 何謂自由意志

自由意志到底是什麼？小弟認為，這是神給予我們擁有像神一樣的特質。

翻開聖經
```
創世記 1:26
神說：「我們要照著我們的形像、按著我們的樣式造人，使他們管理海裡的魚、空中的鳥、地上的牲畜和全地，並地上所爬的一切昆蟲。」
創世記 2:7
耶和華神用地上的塵土造人，將生氣吹在他鼻孔裡，他就成了有靈的活人，名叫亞當。
```

神造人的時候，不僅賦予我們管理萬物的權力 ，更在我們的鼻孔吹了一口氣，
讓我們成為「有靈的活人」 。
這份靈性讓我們具備了與神對話、做出選擇的能力 。
自由意志本身就是這個形像中最核心的特質之一，因為神本身擁有絕對的自由與主權 。

為了讓大家明白這份自由有多可貴，我們來看看聖經中一個很強烈的對比：
神掌控活物的一舉一動，但對於「人心」，祂卻給了完全的自由。

- **1.天氣與時間**：
	神命令降下大雨，洪水就泛濫在地（創世記 7:11-12）；
	神要日頭停住，太陽就在天空中停了一整天（約書亞記 10:12-13）。

- **2.活物的行動**：
	神阻止亞伯拉罕獻上他唯一的兒子時，那隻公羊就乖乖出現在樹叢裡，角剛好被扣住（創世記 22:13）；
	神要管教先知，大魚就準時出現吞下約拿（約拿書 1:17）；
	神甚至能讓不能說話的驢子開口阻止先知（民數記 22:28）。

- **3.靈界的權柄**：
	那些有能力的天使也是聽命行事的使者（詩篇 103:20）；
	甚至是墮落的魔鬼，也必須在神的許可跟計畫底下才能動彈（約伯記 1:12、馬太福音 8:28-32）。

在這些例子裡面，你看不到受造物有任何討價還價的餘地。

可是，轉過頭來看我們人，神雖然掌控萬有，唯獨對「我們這顆心」，祂給予了極大的自由，幾乎不強迫、也不直接干涉。

- **亞伯拉罕的選擇**：
	神絕對有能力強迫羊出現在祭壇，但祂卻不強迫亞伯拉罕獻祭。
	亞伯拉罕當初決定離開拜偶像的故鄉，那跨出家門的第一步，
	是他運用神給的自由意志，對神的呼召給予的回應（創世記 12:1-4） 。

- **保羅的歸順**：
	大家看保羅在大馬色路上的經歷。神可以用大光擊倒保羅的身體，這展現了神的能力；
	但神並沒有直接「洗腦」保羅，讓他立刻變好。神是在等待保羅自己開口問那一句：
	「主啊，我當做什麼？」（使徒行傳 22:10） 。

這就是我想告訴大家的：神不想要一群只會聽指令的機器人，
祂要的是一個能與祂對話、心甘情願愛祂的「兒女」 。

#### 原罪與枷鎖

既然神這麼尊重我們，給了我們高貴的選擇權，但問題來了：為什麼我們最後偏偏選到了「不自由」呢？
為什麼這份原本用來榮耀神的禮物，最後會變成我們生命中沈重的枷鎖？

這就要從人類第一次「選錯了」開始說起。神在伊甸園給了亞當跟夏娃一份「生命規約」，唯一的要求就是不要吃善惡果 。
但在那次關鍵的選擇中，他們在自由意志中失敗了，這次失敗讓「罪」進入了人的靈魂，成為了我們所有痛苦的源頭 。

我們可以從亞當和夏娃吃完果子後的「第一現場」，看看原罪是怎麼破壞我們的生命：

#### 1. 遮羞與掩蓋：

聖經說他們吃完後，「眼睛就明亮了」 。這不是說他們以前是瞎子，而是心裡「明白」了，卻是往不好的方向明白 。

他們突然發現自己赤身露體，第一反應就是拿無花果樹的葉子編裙子擋起來 。
這不只是身體害羞，而是內心感到不堪羞恥，當時明明就夫妻二人而已，以前也不會覺得這有什麼，
所以可見人遠離了神，原本那種聖潔、坦蕩的自信就不見了，
取而代之的是想把醜陋藏起來的虛榮 。我們現在是不是也常這樣？用很多謊言、藉口（像無花果葉一樣）來遮掩自己的不堪 。

#### 2. 恐懼與逃避：

以前的時候，他們最期待在園子裡跟神散步；但犯罪後，天起涼風...一聽到神的聲音，反應竟然是「躲起來」 。
這就是跟神關係斷掉了 。經上說凡作惡的都「恨光」，因為恐怕自己的行為受責備 。
當我們心裡有罪，我們不再覺得神是慈愛的父親，反而覺得祂是來抓人的警察 。
這種不敢面對神、怕被罵的不平安，就是原罪帶來的靈魂重擔 。

#### 3. 推卸責任：

這是罪性最醜陋的一面。當神問亞當是不是偷吃了，亞當竟然沒有認錯，反而開始「大甩鍋」 。
亞當說：「是你賜給我、與我同居的那女人給我的。」 亞當第一次看到他的太太，
他很歡喜說這是我骨中的骨、肉中的肉，但現在要被神歸咎責任了，
他不但把錯推給自己的太太，還暗中怪神「雞婆」造了個女人害他 。
沒犯錯之前是骨中的骨、肉中的肉非常寶貴的...犯錯之後，變成了肉骨茶麵，倒掉都不心疼的...
夏娃也馬上說：「是蛇引誘我。」
這就是被罪控制後的反應我們永遠不會檢討自己，只會找理由 。
功勞大家搶，出事了就說「我早就跟你說過不要做了」 。

這份壞掉的自由意志，讓我們從神尊貴的兒女，變成了被情緒、被恐懼、被慾望牽著走的奴隸 。

犯罪之後，不只是我們的心變了，連整個世界都變得「不友善」了：

以前在伊甸園，工作是「修理看守」，那是一種享受 。但犯罪之後，神對亞當說：地必為你的緣故受咒詛

- 地長出了荊棘跟蒺藜
- 汗流滿面才能餬口
- 女人要加增懷胎的苦楚

我們常說「人非聖賢、孰能無過」，但其實我們是被體內的「七情六慾」給綁架了。

吃了善惡果之後，我們的眼睛確實明亮了，但麻煩的是，我們「知道」什麼是善，卻沒力氣去做；
「知道」什麼是惡，卻忍不住去行...

翻開聖經
```
加拉太書 5:19-21
19 情慾的事都是顯而易見的，就如姦淫、污穢、邪蕩、
20 拜偶像、邪術、仇恨、爭競、忌恨、惱怒、結黨、紛爭、異端、
21 嫉妒（有古卷在此有：凶殺二字）、醉酒、荒宴等類。我從前告訴你們，現在又告訴你們，行這樣事的人必不能承受神的國。
```

保羅曾說
```
羅馬書 7:18-19、24
18 立志為善由得我，只是行出來由不得我。
19 故此，我所願意的善，我反不做；我所不願意的惡，我倒去做。
24 我真是苦啊！誰能救我脫離這取死的身體呢？
```

神雖然讓我們面對勞苦，卻沒有打算讓我們『白白受苦』。
在創世記祂在那件皮衣裡，藏了一個計畫一個預表，
有一位要帶領我們這群累得半死的人，給我們安息，
保羅也給出了答案，也是小弟這次想要探討的，
我們要如何得到這個真自由，找回那份失落的自由。

## 三、 找回自由

剛才我們說到，人被罪制伏了，就像掉進泥潭裡，
單靠自己是不行的，結果只是徒勞無功，

很多人以為找回自由要靠「積功德」，這就像是欠了幾千萬，
卻想靠每天扶老太太過馬路來還債，那也是行不通的 。
這不是我們可以靠著多做好事換來的，
如果我們可以靠自己變好，那耶穌就不用死在十字架上了。

#### 基督的犧牲

那神是怎麼幫我們找回自由的呢？其實在人類犯罪的那一刻，神就已經開始動工了 。

#### 1. 皮衣的預表：救恩的起點

在創世記第三章最後，神做了一個很有溫度的動作：祂親手用「皮子」做衣服給亞當夏娃穿 。

- **犧牲生命**：
	大家想一想，要有皮子做衣服，就一定要殺掉動物，一定要有生命流出血來、犧牲生命
- **神的慈愛**：
	神雖然罵了他們，甚至把他們趕出伊甸園，但神的心是很痛的 。
	祂不忍心看亞當只用那些會枯乾的樹葉遮羞，所以祂親自預備了皮衣 。
	這就是聖經中救恩計畫的第一次呈現 。

#### 2. 耶穌基督：真正救贖的代價

這件皮衣預表的，就是兩千年後的耶穌基督 。

- **耶穌的名字**：
	當初天使告訴約瑟，要給瑪利亞生的兒子取名叫「耶穌」，因為這個名字的意思就是「救主」，
	祂要將自己的百姓從罪惡中拯救出來 。
    
- **道成肉身的動作**：
	神愛我們不是光說不練，祂是親自「道成肉身」來到世上 。
	耶穌就是那個「神的羔羊」，祂用自己的生命代替了我們的生命，流出寶血來遮蓋我們的罪 。
    
- **洗淨靈魂的中毒**：
	這份救恩不是把我們這台「中毒的電腦」隨便修修就好，
	而是主耶穌用祂的生命（神的羔羊）代替了我們，流出寶血洗淨我們的檔案，給我們一個全新的生命系統 。

#### 3. 從「奴隸」變成「神的子民」

所以，找回自由其實就是一場「身分的大轉變」 。

- **國度的搬家**：主耶穌要帶領我們從「罪惡的國度」移民到「神的國度」 。
    
- **主權的移交**：
	以前我們是被罪、被壞習慣、被魔鬼牽著走的「奴僕」 。
	但現在我們換了一個主人，只要我們願意順服神的主權，這份真自由就開始在我們心裡發芽 。

#### 蔡弟兄的見證

剛才我們提到保羅說他「真是苦啊」，那種知道什麼是對的卻做不到、不想做的惡卻偏要去做的掙扎，
有一位蔡弟兄在年輕的時候體會得最為深刻 。

#### 1. 追求自由的代價：淪為毒品的奴隸

蔡弟兄年輕的時候，跟很多少年人一樣，覺得「只要我喜歡，有什麼不可以」就是自由 。為了追求快樂、追求刺激，他開始嘗試吃所謂的「快樂丸」（搖頭丸） 。

- **從好奇到成癮**：剛開始覺得自己只是在玩玩，但沒想到這份「自由」很快就變成了最恐怖的監獄 。
    
- **身體的殘破**：後來他不只是吃藥，還發展到注射毒品 。大家想像一下那個畫面，他來到教會時，雙手手臂上密密麻麻全是針孔，多到沒地方再扎針 。
    
- **意志的挫敗**：蔡弟兄當時有沒有想過要戒？當然有！他知道毒品在毀掉他的命，但他發現自己「停不下來」 。這就是聖經說的：「人被誰制伏，就是誰的奴僕」 。
    

#### 2. 神的呼喚：在園子裡的尋找

就在蔡弟兄覺得自己快要爛在泥潭裡、徹底沒救的時候，神沒有放棄他 。

- **主動的關心**：神透過教會的信徒去關心他、找回他 。就像神在伊甸園呼喊亞當一樣，神也在呼喚蔡弟兄：「你在哪裡？你要不要回來？」 。
    
- **來到神面前**：蔡弟兄帶著那雙滿是針孔的手，回到了神的家 。他承認自己這台電腦已經徹底中毒黑屏了，靠自己修不好 。
    

#### 3. 聖靈的火：徹底的翻身與釋放

這是見證最關鍵的部分。蔡弟兄知道，如果只靠他那份「壞掉的自由意志」，他一踏出教會大門就會再去找毒販 。

- **迫切的求救**：他開始在神面前流淚禱告，他求的不是發大財，而是求神給他「聖靈」 。
    
- **聖靈的力量**：當他求到聖靈後，那份來自神的能力進到了他的生命裡 。保羅說的「生命聖靈的律」在他身上發生了化學反應，這股強大的動力竟然壓過了毒癮的誘惑 。
    
- **神人合作的自由**：蔡弟兄發現，以前戒不掉的毒，現在竟然能打從心底說「不」 。這不是他變強了，而是他換了一個主人，主耶穌在他心裡坐鎮，讓他在面對惡事時擁有了「拒絕的權利」 ，有能力有勇氣去拒絕。


弟兄姊妹，看著蔡弟兄那雙曾經滿是針孔的手，我們就能明白什麼叫『被制伏』 。

很多時候，我們雖然沒有吸毒，但我們是不是也被某些壞習慣、某些負面情緒給制伏了？
我們想改，卻改不掉；我們想溫柔，卻忍不住發脾氣 。

蔡弟兄的見證告訴我們一個祕訣：自由，是神與人合作的結果 。

神預備了救恩這件『皮衣』，也預備了『聖靈』與我們同在，加添我們力量 。
但我們必須像蔡弟兄一樣，運用我們最後一點自由意志，
向神開口呼求：『主啊，救我！』 。

當你願意跨出這一步，主耶穌就要將你從罪惡中拯救出來，讓你找回那份失落已久的、真正的自由 。

## 五、 結語：神的呼召
