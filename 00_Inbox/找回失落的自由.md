## 一、 引言：我們真的自由嗎？ 

哈利路亞～各位弟兄姊妹，平安。 今天想跟大家分享的主題是：「找回失落的自由」。 
大家聽到這個題目可能會想，怎麼會定這個主題呢？難道我們現在不自由嗎？

提到自由，很多人可能會想到匈牙利詩人裴多菲的一首名詩：
```
生命誠可貴，愛情價更高，若為自由故，兩者皆可拋
```

這句詩深刻地描繪出，在自由面前，連生命與愛情都可以放下。
縱觀人類的歷史，幾乎就是一個爭取自由的過程。

而回到我們現在的生活，從台灣的法律來看，憲法確實保障了我們許多權利：
言論自由、信仰自由、居住跟遷徙的自由。
簡單來說，法律定義的自由就是：「只要不違法、不侵犯別人，我想做什麼就可以做什麼，沒有人可以隨便限制我。」

如果單從這個角度來看，我們確實活在一個極度自由的國度。我們想吃什麼、去哪裡玩、要選擇什麼樣的信仰，
都可以憑著神造人的時後所賜給我們的「自由意志」來決定 。

但是，法律保障了我們身體和行為的自由，我們的心靈，真的自由嗎？

翻開聖經
```
彼得後書 2:19
他們應許人得以自由，自己卻做敗壞的奴僕，因為人被誰制伏，就是誰的奴僕。
```

親愛的弟兄姊妹們，其實我們今天遇到的這種『假自由』，在兩千年前的教會就發生過了。

當時的使徒彼得，面對教會裡的一群『假師傅』。
這些人到處跟信徒開空頭支票說：「信耶穌了，我們自由啦！肉體想做什麼都可以，盡情去享受吧！」
這聽起來，是不是跟我們現在社會常聽到的「只要我喜歡，有什麼不可以」？

但彼得直接戳破了這個謊言。彼得的意思是，這些人嘴裡喊著自由的口號，
實際上他們根本控制不了自己的慾望。他們被貪婪制伏、被情慾制伏，說穿了，他們其實是慾望的奴隸！

這句話套用在我們現代人身上，簡直如出一轍。
當我們以為放縱自己去沉迷網路與社群、去抽菸酗酒是一種自由的時候，
其實我們已經被這些習慣給『制伏』了，不知不覺中，我們成了這些壞習慣的奴僕。

這就是我今天想要探討的核心。神明明給了我們尊貴的自由意志，為什麼我們最後會落入這種被制伏、不自由的狀態呢 ？
其實，這一切都要回到我們「人性的軟弱」來談。
接下來，我們就來看看，這種軟弱是如何讓我們濫用了自由意志，以及現代的文化跟風俗又是怎麼轄制我們的……

## 二、 人性的軟弱

#### 何謂自由意志

自由意志到底是什麼？小弟認為，這是神給予我們擁有像神一樣的特質。

翻開聖經
```
創世記 1:26
神說：「我們要照著我們的形像、按著我們的樣式造人，使他們管理海裡的魚、空中的鳥、地上的牲畜和全地，並地上所爬的一切昆蟲。」
創世記 2:7
耶和華神用地上的塵土造人，將生氣吹在他鼻孔裡，他就成了有靈的活人，名叫亞當。
```

神造人的時候，不僅賦予我們管理萬物的權力 ，更在我們的鼻孔吹了一口氣，
讓我們成為「有靈的活人」 。
這份靈性讓我們具備了與神對話、做出選擇的能力 。
自由意志本身就是這個形像中最核心的特質之一，因為神本身擁有絕對的自由與主權 。

為了讓大家明白這份自由有多可貴，我們來看看聖經中一些很強烈的對比：
神掌控宇宙萬物的一舉一動，但對於「人心」，祂卻給了完全的自由。

- **1.天氣與時間**：
	挪亞的時代，神命令降下大雨，洪水就泛濫在地（創世記 7:11-12）；
	以色列人在征戰的時候，神要日頭停住，太陽就在天空中停了一整天（約書亞記 10:12-13）。

- **2.活物的行動**：
	神阻止亞伯拉罕獻上他唯一的兒子時，那隻公羊就乖乖出現在樹叢裡，角剛好被扣住（創世記 22:13）；
	神要管教先知，大魚就準時出現吞下約拿（約拿書 1:17）；
	神甚至能讓不能說話的驢子開口阻止先知（民數記 22:28）。

- **3.靈界的權柄**：
	那些有能力的天使也是聽命行事的使者（詩篇 103:20）；
	甚至是墮落的魔鬼，也必須在神的許可跟計畫底下才能動彈（約伯記 1:12、馬太福音 8:28-32）。

在這些例子裡面，你看不到受造物有任何討價還價的餘地。
可是，轉過頭來看我們人，神雖然掌控萬有，唯獨對「我們這顆心」，
祂給予了極大的自由，幾乎不強迫、也不直接干涉。

- **亞伯拉罕的選擇**：
	神絕對有能力直接把羊變在祭壇上，甚至祂大可直接『控制』亞伯拉罕的身體去獻祭他兒子。
	但對於人心，神給了選擇權。

	面對獻以撒這個極難的考驗，神下了命令，卻留給了亞伯拉罕整整三天的路程。
	在這三天裡，亞伯拉罕隨時可以轉頭放棄、隨時可以對神說『不』。
	神沒有把他當成聽指令的機器人，而是讓他經歷內心的掙扎後，
	因著對神的信任，自己『選擇』了順服。」

- **保羅的歸順**：
	大家看保羅在大馬色路上的經歷。神可以用大光擊倒保羅的身體，這展現了神的能力；
	但神並沒有直接「洗腦」保羅，讓他立刻醒悟。神是在等待保羅自己開口問那一句：
	「主啊，我當做什麼？」（使徒行傳 22:10） 。

這就是我想告訴大家的：神不想要一群只會聽指令的機器人，
祂要的是一個能與祂對話、心甘情願愛祂的「兒女」 。

#### 原罪與轄制

既然神這麼尊重我們，給了我們高貴的選擇權，但問題來了：為什麼我們最後偏偏選到了「不自由」呢？
為什麼這份原本用來榮耀神的禮物，最後會變成我們生命中沈重的枷鎖？

這就要從人類第一次「選錯了」開始說起。神在伊甸園給了亞當跟夏娃一份「生命規約」，唯一的要求就是不要吃善惡果 。
但在那次關鍵的選擇中，他們在自由意志中失敗了，這次失敗讓「罪」進入了人的靈魂，成為了我們所有痛苦的源頭 。

我們可以從亞當和夏娃吃完果子後的「第一現場」，看看原罪是怎麼破壞我們的生命：

#### 1. 遮羞與掩蓋：

聖經說他們吃完後，「眼睛就明亮了」 。這不是說他們以前是瞎子，而是心裡「明白」了，卻是往不好的方向明白 。

他們突然發現自己赤身露體，第一反應就是拿無花果樹的葉子編裙子擋起來 。
這不只是身體害羞，而是內心感到不堪羞恥，當時明明就夫妻二人而已，以前也不會覺得這有什麼，
所以可見人遠離了神，原本那種聖潔、坦蕩的自信就不見了，
取而代之的是想把醜陋藏起來的虛榮 。我們現在是不是也常這樣？用很多謊言、藉口（像無花果葉一樣）來遮掩自己的不堪 。

#### 2. 恐懼與逃避：

以前的時候，他們最期待在園子裡跟神散步；但犯罪後，天起涼風...一聽到神的聲音，反應竟然是「躲起來」 。
這就是跟神關係斷掉了 。經上說凡作惡的都「恨光」，因為恐怕自己的行為受責備 。
當我們心裡有罪，我們不再覺得神是慈愛的父親，反而覺得祂是來抓人的警察 。
這種不敢面對神、怕被罵的不平安，就是罪性帶來的靈魂重擔 。

#### 3. 推卸責任：

這是罪性最醜陋的一面。當神問亞當是不是偷吃了，亞當竟然沒有認錯，反而開始「大甩鍋」 。
亞當說：「是你賜給我、與我同居的那女人給我的。」 亞當第一次看到他的太太，
他很歡喜說這是我骨中的骨、肉中的肉，但現在要被神歸咎責任了，
他不但把錯推給自己的太太，還暗中怪神「雞婆」造了個女人害他 。
沒犯錯之前是骨中的骨、肉中的肉非常寶貴的...犯錯之後，變成了肉骨茶麵，倒掉都不心疼的...
夏娃也馬上說：「是蛇引誘我。」
這就是被罪控制後的反應我們永遠不會檢討自己，只會找理由 。
功勞大家搶，出事了就說「我早就跟你說過不要做了」 。

這份壞掉的自由意志，讓我們從神尊貴的兒女，變成了被情緒、被恐懼、被慾望牽著走的奴隸 。

犯罪之後，不只是我們的心變了，連整個世界都變得「不友善」了：

以前在伊甸園，工作是「修理看守」，那是一種享受 。但犯罪之後，神對亞當說：地必為你的緣故受咒詛

- 地長出了荊棘跟蒺藜
- 汗流滿面才能餬口
- 女人要加增懷胎的苦楚

不只是這個世界改變了，變得讓人生活得很辛苦，我們看到人內心的醜惡也敗壞了我們生命，
聖經告訴我們，很多人雖身處在自由的國度，無拘無束，但是有一股力量在控制人、轄制人，使人十分不自由。

翻開聖經
```
以弗所書 2:1-3
1 你們死在過犯罪惡之中，他叫你們活過來。
2 那時，你們在其中行事為人，隨從今世的風俗，順服空中掌權者的首領，就是現今在悖逆之子心中運行的邪靈。
3 我們從前也都在他們中間，放縱肉體的私慾，隨著肉體和心中所喜好的去行，本為可怒之子，和別人一樣。
```

這段聖經告訴我們幾個重點：很多人看起來是自由，其實是放縱；很多人看似自由，
其實是受到空中屬靈氣惡魔的一種轄制。
使我們可能有善心卻行不出來，不想做的、不該做的卻衝動行事...無可奈何
保羅也有這樣的體悟

```
羅馬書 7:18-19、24
18 立志為善由得我，只是行出來由不得我。
19 故此，我所願意的善，我反不做；我所不願意的惡，我倒去做。
24 我真是苦啊！誰能救我脫離這取死的身體呢？
```

## 三、 找回自由

既然我們發現自己掉進了罪的泥潭中，單靠意志力行不通，那我們到底該怎麼辦才能找回真正的自由？

小弟認為，真正的自由其實很簡單，就是「不再被罪惡與誘惑所挾制」 。 我們常會有一種錯覺，
覺得信主後「這也不能做、那也不能做」，好像變得很不自由。但我們回頭想想創世記的伊甸園，當時神只有一種果子規定不能吃，
其他的果子，甚至是生命樹上的果子，亞當夏娃其實都能隨便吃 。

為什麼人偏偏要去吃那唯一不能吃的善惡果？這就是仇敵的詭計：牠讓我們只盯著那 1% 的限制，
卻忘了神給了我們 99% 廣大的自由 。
當然今非昔比，現在的情況更複雜，但其實該念是一樣的，
那些犯罪、不該做的事，本來就對生命沒有任何正面的意義，只會帶給我們損傷 。

#### 1. 曉得真理：自由的「軌道」

主耶穌親自教導我們脫離罪惡的秘訣：

翻開聖經
```
約翰福音 8:32、34、36
32 你們必曉得真理，真理必叫你們得以自由。
34 耶穌回答說：「我實實在在地告訴你們：所有犯罪的就是罪的奴僕；
36 所以天父的兒子若叫你們自由，你們就真自由了。
```

我們今天要得以自由，不是為非作歹、沒有規範。真正的自由就好像一輛火車行駛在鐵軌上面，是有規範、有方向、有目標的。
我們的人生也是一樣，到底該做什麼、往哪個方向，聖經的真理告訴我們很清楚。有人認為相信耶穌、成為基督徒很不自由，
好像這不能做、那不能做；但其實真正的自由是「有所為，有所不為」，不該做、不當做的就不去做，該做的就努力去實行。
透過聖經的真道，就像法律、就像父母、老師的交代，聖經也交代得很清楚。

#### 2. 得救的智慧

聖經也提醒我們要及早認識神的話：

```
提摩太後書 3:15
並且知道你是從小明白聖經，這聖經能使你因信基督耶穌，有得救的智慧。
```

這世上有很多種智慧，比如學識、發明或待人接物的智慧；但聖經給我們的是最重要的「生命得救的智慧」 。
它幫助我們辨別哪條路通往永生，哪條路通往滅亡 。

#### 見證分享

很多人明明知道菸、酒、毒品不好，但偏偏去行。明明知道抽菸不好，菸盒上也提醒有害健康，
但因為受到菸癮控制還是做了；很多人知道酗酒不好，但心情不好就喝，把肝都喝壞了。甚至有人受到毒品的控制。

有一位蔡弟兄，在年輕時後很不快樂。朋友告訴他：「這裡有一顆藥你一吃，絕對能得到快樂、得到自由」。
當他這顆藥一吃，天旋地轉，好像得到了釋放。但自此之後他就染上了毒品，甚至越吃越重，
從用吃的改用打的，打到最後兩隻手都布滿了針孔。他不敢讓人家知道，原本想要放棄人生；他知道吸毒不對，但無奈戒不掉。

他有個鄰居是真耶穌教會的信徒，邀請他來到教會。起初他不願意，
但在盛情邀約下他來慕道了。第一次聚會居然感受到一股力量一直在感動他的心，
佈道會結束回程路上，他就告訴我們弟兄：「其實我在吸毒。」這話嚇了弟兄一大跳，但弟兄鼓勵他：「你要靠著聖靈幫助你戒掉毒癮」。

真的，他有信心，禱告求聖靈，神賜給他聖靈，
當毒癮一來的時候他就拼命禱告，有時一天禱告三、四個小時，
當他迫切禱告，毒癮就慢慢退去。感謝主，靠著聖靈的幫助，他真的戒掉了毒癮，
所以聖經所說的是真的，透過聖靈的幫助，能讓我們真正得著釋放，就連吸毒的都能在主裡得享真正的自由。

蔡弟兄的見證告訴我們，自由不是「隨心所欲」，而是當你知道什麼是正確的事時，
你擁有「能力」去實踐它 。這份能力，就是來自聖靈。

翻開聖經：
```
羅馬書 8:2
因為賜生命聖靈的律，在基督耶穌裡釋放了我，使我脫離罪和死的律了。
```

這就是關鍵！我們體內原本有一個「罪和死的律」，讓我們像被磁鐵吸住一樣一直往壞處走；
但聖靈是另一個更強大的「生命聖靈的律」，祂能把我們從罪的吸引力中「釋放」出來 。

```
加拉太書 5:16
我說，你們當順著聖靈而行，就不放縱肉體的情慾了。
```

親愛的弟兄姊妹，這份真實的信仰不是要給我們一堆限制跟規定，
而是要讓我們成為神的孩子，不再是無依無靠、隨便被欺負的孤兒 。
只要我們歸入耶穌基督的名下，靠著聖靈的幫助，我們就能從心裡做出對的選擇，
找回那份失落的、真正的自由 。

## 四、 找回自己

相信大家一開始信主的時候，都是愛神的、有信心的。但是人生要經歷的磨練真的很多，
我們常說「人非聖賢，孰能無過」，在軟弱的時候，我們可能也會遇到挫折、失敗、甚至跌倒。

跌倒之後，最可怕的往往不是那個過犯本身，而是我們心裡會產生一種恐懼，
覺得：「我完蛋了，難道我再也沒有辦法坦然地面對神了嗎。」

但小弟想告訴大家，真實的信仰從來不是一條用來綁死我們的繩子，
而是為了讓我們從「罪的愧疚與恐懼」中獲得真正的自由。**跌倒不可怕，可怕的是你以為自己再也回不去了。**

沒有人是完美的。每個人在言語、行為或思想上，多少都會做一些讓自己後悔的事。
我們來看看聖經裡兩個非常經典的例子：

#### 大衛的過犯

舊約裡有一位大家很熟悉的英雄，就是大衛王。
聖經毫不掩飾地記錄了他人生中最黑暗的一頁。
這告訴我們一件事：**不管你信仰多久、信心多強，只要一不小心，隨時會淪為罪的奴隸。**

**1. 從自由變成了放縱**

大衛的跌倒，不是發生在戰場上，而是發生在「太過安逸」的時候。

- **該在場的時候不在場**：
	聖經說那時是「君王出戰之時」，大衛卻留在耶路撒冷睡到太陽西下，
	在王宮的平頂上散步（撒母耳記下 11:1-2）。
- **眼目的情慾**：
	他看見了拔示巴。這時候他的自由意志做出了一個錯誤的導向，
	他把原本該用來管理國家的「權力」，拿來滿足自己的「私慾」，
	找人調查她，還把她接到皇宮與有夫之婦同寢。

**2. 罪的連鎖反應

大衛犯罪後，他的狀態跟亞當一模一樣，就是想要「遮掩與逃避」。

- **大甩鍋與陰謀**：
	為了掩蓋拔示巴懷孕的事實，他把忠心的烏利亞從前線召回，
	希望烏利亞回家跟太太同寢，好把孩子賴在他身上。
    
- **借刀殺人**：
	結果烏利亞太正直，竟然不肯回家，
	大衛看計畫不成，就動了殺心，寫信給約押，藉著敵人的手殺了烏利亞。

> **這就是罪的枷鎖**：
> 大衛原本是自由的君王，但為了遮掩那個「惡」，他被迫變得更惡。
> 他已經不再是神國的代理人，而是被「恐懼」與「名聲」給綁架的囚犯。

**3. 靈魂被光照**

神有放棄放棄大衛嗎，並沒有...祂差遣了先知拿單去點醒他。

- **貼切的比喻**：
	拿單講了一個「富戶搶窮人小羊」的故事，大衛聽了義憤填膺，說這人該死。
    
- **戳破偽裝**：
	拿單對大衛說：「你就是那人！」（撒母耳記下 12:7）。
	那一刻，我相信大衛是知道他沒招了，因為他知道這是神在告誡他，
	必須要在神面前坦然面對。

**4. 憂傷痛悔的心**

大衛跟亞當最不一樣的地方，是他沒有說「都是那女人害我的」。
他選擇了坦然面對，寫下了痛悔的詩篇《詩篇》51 篇。

親愛的弟兄姊妹，神原諒大衛了嗎？原諒了。
但神也是公義的，大衛雖然得到赦免，他的人生依然承擔了沉重的代價，
家裡經歷了許多風風雨雨。
但這部份我不想說太多...

#### 浪子回頭

這是一個很經典的故事，記載於 《路加福音》第 15 章 11-32 節

**1. 索取家產：輕看分身與自由**

小兒子對父親說：「請把我就得的家產分給我。」在當時的文化，這等同於詛咒父親早死。
但他不想再受家裡的約束，渴望「無拘無束」的自由 。父親沒有強留，而是尊重他的選擇，分了家產。

**2. 揮霍無度：淪為罪的奴僕**

他帶著錢去了遠方，在那裡「任意放蕩，浪費資財」。
正如之前所提到的，當人以為可以隨心所欲、不聽勸誡時，其實並不是真正的自由，而是「做了敗壞的奴僕」 。
當他耗盡一切後，當地又遇上大饑荒，他陷入了極度的窮困。

**3. 窮途末路：失去尊嚴的掙扎**

為了生存，這個猶太青年竟然去替人「放豬」（對猶太人來說，豬是不潔的，這是極大的羞辱）。
他餓到想吃豬食用的豆莢，卻沒人給他吃。

**4. 醒悟回轉：這就是「回家的勇氣」**

聖經描述他「醒悟過來」。他想起父親家裡連雇工都有充足的糧食，而自己卻在這裡快要餓死。
他謙卑下來，預備了一段話想對父親說：

> 「父親！我得罪了天，又得罪了你；從今以後，我不配稱為你的兒子，把我當作一個雇工吧！」

他不敢奢求恢復身分，只求能待在父親身邊 。

**5. 父親的愛：天天守望的憐憫**

這是故事最動人的地方：「相離還遠，他父親看見，就動了慈心，跑去抱著他的頸項，連連與他親嘴。」

- **預先的等待：** 
	父親能「遠遠看見」，難道是剛好路過嗎？
	應該不是的，是父親是天天站在門口望著，期盼他的孩子回來。
- **不計前嫌：** 
	父親還不等他把認罪的話說完，就吩咐僕人拿「最好的袍子」（象徵身分恢復）給他穿，
	把「戒指」（象徵權柄）戴在他手上，把「鞋」（象徵自由人，奴隸是不穿鞋的）給他穿上。
- **慶祝重生：** 
	父親宰了肥牛犢，宣布：「我這個兒子是死而復活，失而又得的。」


親愛的弟兄姊妹，我們把大衛跟浪子放在一起看

他們兩個人也不同也有相同的地方，一個在皇宮的屋頂上，一個在骯髒的豬圈裡，
但他們當時都「失去了自己」**。大衛忘了自己是神所膏立的王，浪子忘了自己是尊貴的兒子。

但是，當大衛流下痛悔的眼淚，當浪子轉身踏上回家的路，神所做的第一件事，不是拿鞭子打他們，
而是**把那件代表神兒女身分的「袍子」，重新披在他們身上。**

這就是小弟今天想跟大家分享的其中一點：**我們來到教會，不是為了比賽誰比較完美。**

主耶穌曾說：「康健的人用不著醫生，有病的人才用得著；我來本不是召義人，乃是召罪人。」
我們沒有一個人是靠著自己完美的行為稱義的，我們都是靠著主耶穌的寶血才得以站在這裡。
既然大家都會軟弱，那就**沒有任何人有資格去定誰的罪**，
我們也不該定別人的罪。

如果你現在正處在低潮，覺得自己曾經遠離神，覺得自己像那個浪子一樣不配；
或者你曾受過挫折，對服事失去了熱情。

真正的自由，是「找回你在神眼中的價值」**。

讓我們重拾信心，不要因為曾經的軟弱就躲在黑暗裡。我們不需要跟別人比較，
只要在我們**能力所及的地方**，努力為神作工。哪怕你覺得自己力量很小；
哪怕你只是在台面下默默付出、排排椅子、打掃環境；
哪怕你只是在心裡默默為教會代禱，神都看重，神都紀念！

願我們都能在神的愛中，找回那個失落已久的自己，勇敢地站起來，為這位愛我們的神，繼續發光發熱。

## 五、 結語

最後，小弟想用一段經文，來作為今天分享的總結。

翻開聖經：
```
加拉太書 5:13
弟兄們，你們蒙召是要得自由，只是不可將你們的自由當作放縱情慾的機會，總要用愛心互相服事。
```

我們今天談了很多關於「自由」的掙扎。從伊甸園裡被誤用的自由，
到被現代社會各種慾望與壞習慣轄制，再到大衛和浪子的迷失。我們看見了人性的軟弱，
但也看見了神永不放棄的慈愛。

感謝神，因著耶穌基督的救恩與聖靈的幫助，我們已經找回了那份最寶貴的兒女身分。

保羅在這裡提醒我們，**真正的自由，不是拿來放縱自己的私慾，而是讓我們終於擁有了「去愛」的能力。** 
當我們不再被罪惡與恐懼綁架時，我們才能像經文所說的，用這份得來不易的真自由，去「用愛心互相服事」。

不論你過去經歷了什麼挫折，或是曾在信仰的路上迷了路，
會離開的都是我們，我們的天父從來沒離開過。
讓我們徹底脫下那件名為愧疚感的沉重枷鎖，穿上神賜給我們的榮耀衣袍。
用我們真實的生命，在神的家裡作一個真正自由、充滿喜樂的兒女。

願一切榮耀、頌讚都歸給天上的真神，平安福氣賜給在座的每一位弟兄姊妹。
