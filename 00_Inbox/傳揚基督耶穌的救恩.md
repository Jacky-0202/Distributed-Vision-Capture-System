1. **使命的內容與範圍：** 定義了教會要傳什麼、傳給誰。

```
路加福音 24:47-48
47 並且人要奉他的名傳悔改、赦罪的道，從耶路撒冷起直傳到萬邦。
48 你們就是這些事的見證。
```

2. **執行的能力與動力：** 強調傳福音不是靠人的熱心，而是聖靈的能力。

```
使徒行傳 1:8
但聖靈降臨在你們身上，你們就必得著能力，並要在耶路撒冷、猶太全地，和撒馬利亞，直到地極，作我的見證。
```

3. **永恆的藍圖與定位：** 說明教會存在於世，是為了彰顯神創世以來的智慧。

```
以弗所書 3:10-11
10 為要藉著教會使天上執政的、掌權的，現在得知神百般的智慧。
11 這是照神從萬世以前，在我們主基督耶穌裡所定的旨意。
```

#### 心得

教會並非偶然成立的團體，而是神從萬世以前就預定好的「救恩傳播中心」。

主耶穌在地上先選召門徒（教會的雛形），接著在完成十字架救恩後，差遣聖靈降臨賜下「能力」。這股能力唯一的目的，就是要教會背負起福音使命——把悔改赦罪的道傳向全世界。教會若只顧著內部聚會而忘了向外宣教，就失去了神建立教會最重要的功能。我們是被揀選的見證人，傳福音是教會不可推卸的神聖託付。
