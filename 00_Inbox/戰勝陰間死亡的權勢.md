```
羅馬書 5:12 
這就如罪是從一人入了世界，死又是從罪來的；於是死就臨到眾人，因為眾人都犯了罪。
```

```
羅馬書 6:23
因為罪的工價乃是死；惟有神的恩賜，在我們的主基督耶穌裡，乃是永生。
```

自從始祖犯罪後，全人類都被魔鬼用「罪」與「死」的毒鉤死死地綁架了，
結局就是地獄的刑罰，沒有任何人能靠自己逃脫。世界上沒有任何偉大的組織或機構可以打贏死亡。

```
希伯來書 2:14
兒女既同有血肉之體，他也照樣親自成了血肉之體，特要藉著死敗壞那掌死權的，就是魔鬼，
```

```
哥林多前書 15:55-56
55 死啊！你得勝的權勢在哪裡？死啊！你的毒鉤在哪裡？
56 死的毒鉤就是罪，罪的權勢就是律法。
```

```
馬太福音 16:18
我還告訴你，你是彼得，我要把我的教會建造在這磐石上；陰間的權柄，不能勝過他。
```

但是，主耶穌來到世上建立教會，並賜給了教會一個最核心、最獨特的特權：**勝過陰間與死亡的權柄**。這就是為什麼教會跟我們的「得救」有絕對的關係。離開了教會，我們只能單獨面對必敗的死亡；唯有躲進主耶穌所建立的教會裡，死亡的權勢在我們身上才會徹底失效。
