## 前言

大家看過動漫或英雄電影吧？裡面的大反派不管多強，最後都會被主角打敗。
但在現實世界裡，有一個全人類都打不贏的「終極大魔王」，叫做「死亡」。
不過，主耶穌卻在地上建立了一個連死都害怕的地方，你們知道是哪裡嗎？

---
## 核心經文

```
馬太福音 16:18
我還告訴你，你是彼得，我要把我的教會建造在這磐石上；陰間的權柄，不能勝過他。    
```

#### 分享

自從人類始祖犯罪後，就像不小心下載了病毒一樣，罪性跟原罪進入了世界，而罪帶來的最終結果就是「死亡」 。
聖經告訴我們，魔鬼就像是掌握了死權的綁匪，把所有帶著罪的人都困在地獄的刑罰裡，沒有人可以靠自己逃脫 。

但是，耶穌來到世上建立了「教會」，並且直接賦予了教會一個特權——勝過陰間與死亡的權柄 ！
主耶穌在這裡給了一個絕對的保證：死亡的力量再大，也吞滅不了神的教會。
#### 結論

教會不只是一個週末交朋友、唱詩歌的地方，它是全宇宙唯一能戰勝「死亡」的堡壘 。
這也是為什麼我們能不能得救，跟我們有沒有在教會裡，有絕對的關係 ！

#### 問題

既然聖經說「陰間的權柄不能勝過教會」、「教會能戰勝死亡」，那為什麼我們身邊還是會看到基督徒生病，
甚至面臨肉體的死亡呢？難道是這個「外掛」失效了嗎？

#### 解答

```
啟示錄 2:11
聖靈向眾教會所說的話，凡有耳的，就應當聽！得勝的，必不受第二次死的害。
```

當然不是失效！這裡說的死亡，指的是「靈魂永遠的滅亡（去地獄）」。對於不在教會裡的人，死亡是可怕的終點；
但對於在教會裡的我們來說，肉體的死亡就像是「睡著了」或是「移民搬家」，
指的是一扇通往天國的門。因為我們在教會裡，魔鬼已經沒有權利把我們的靈魂拖進地獄了，這才是真正的「戰勝死亡」！
