「弟兄姊妹，我們來想一個問題：教會，到底是神什麼時候發明的？是不是因為人類在伊甸園犯罪了，神突然覺得『哎呀，糟了，人類壞掉了，我趕快成立一個叫做教會的團體來補救一下』？

絕對不是！教會從來不是神的『備案』（Plan B），它是神一開始就畫好的『藍圖』。我們把時間拉回最一開始，用三個步驟來看神到底在做什麼

#### 第一步：神給人自由意志

一開始在伊甸園，神造人不是造『機器人』。如果神按一個按鈕，我們就會自動敬拜祂，那不叫愛，那叫寫程式。神極其愛我們，所以祂給了人類一個非常尊貴的權利，叫做自由地選擇。神希望我們是『心甘情願』地愛祂、聽祂的話。

```
創世記 2:16 
耶和華神吩咐他說：「園中各樣樹上的果子，你可以隨意吃，
```

#### 第二步：人把自由玩砸了 

但很可惜，我們的始祖亞當和夏娃，濫用了這份自由。神說這棵樹的果子不能吃，他們偏要吃。當人類選擇偏離真理的那一刻，代價就來了——就是死亡，並且被趕出伊甸園，失去了原本那個尊貴、榮耀的身分。

```
創世記 2:17
只是分別善惡樹上的果子，你不可吃，因為你吃的日子必定死。」
```

#### 第三步：終極救援計畫

人類墮落了，那怎麼辦？其實，全知的神早就知道人會軟弱。

保羅說得非常清楚，早在創立世界以前，神就已經設計好了一個救援計畫。這個計畫的載體，就叫做『教會』！神早就決定好，要在這個世界上設立一個避風港，把那些願意悔改的人，重新聚集起來。

```
馬太福音 16:18-19
18 我還告訴你，你是彼得，我要把我的教會建造在這磐石上；陰間的權柄（權柄：原文作門）不能勝過他。
19 我要把天國的鑰匙給你，凡你在地上所捆綁的，在天上也要捆綁；凡你在地上所釋放的，在天上也要釋放。」
```
