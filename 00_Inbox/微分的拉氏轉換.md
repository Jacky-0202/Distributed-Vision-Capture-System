## 🛠️ 實戰解題：$y'' + 12y' + 32y = 32u(t)$

#### 第一步：全面轉換（進入 $s$ 領域）

我們對方程式兩邊同時施加拉氏轉換：

1. $\mathcal{L}\{y''\} = s^2Y(s) - sy(0) - y'(0) = s^2Y(s)$
2. $\mathcal{L}\{12y'\} = 12[sY(s) - y(0)] = 12sY(s)$
3. $\mathcal{L}\{32y\} = 32Y(s)$
4. $\mathcal{L}\{32u(t)\} = 32 \cdot \frac{1}{s} = \frac{32}{s}$ （還記得 $u(t)$ 就是 1 吧？）

**整理後的代數方程：**

$$(s^2 + 12s + 32)Y(s) = \frac{32}{s}$$

---
#### 第二步：代數求解（孤立 $Y(s)$）

我們要找出 $Y(s)$ 是什麼。先把左邊的二次式因式分解：

$s^2 + 12s + 32 = (s+4)(s+8)$

所以：

$$Y(s) = \frac{32}{s(s+4)(s+8)}$$

---
#### 第三步：零件分類（部分分式展開）

現在這是一坨「通分完的濃湯」，我們要用**覆蓋法**把它拆回三個簡單零件：

$$\frac{32}{s(s+4)(s+8)} = \frac{A}{s} + \frac{B}{s+4} + \frac{C}{s+8}$$

1. **求 A (遮住 $s$，代入 $s=0$)**：
    
    $$A = \frac{32}{(0+4)(0+8)} = \frac{32}{32} = 1$$
    
2. **求 B (遮住 $s+4$，代入 $s=-4$)**：
    
    $$B = \frac{32}{(-4)(-4+8)} = \frac{32}{-16} = -2$$
    
3. **求 C (遮住 $s+8$，代入 $s=-8$)**：
    
    $$C = \frac{32}{(-8)(-8+4)} = \frac{32}{32} = 1$$
    

**拆解結果：**

$$Y(s) = \frac{1}{s} - \frac{2}{s+4} + \frac{1}{s+8}$$

---

#### 第四步：還原真相（反拉氏轉換）

現在每一項都是我們認識的「基本零件」了，直接讀取答案：

1. $\mathcal{L}^{-1}\{\frac{1}{s}\} = 1$ (或者寫成 $u(t)$)
2. $\mathcal{L}^{-1}\{\frac{2}{s+4}\} = 2e^{-4t}$
3. $\mathcal{L}^{-1}\{\frac{1}{s+8}\} = e^{-8t}$

**最終解：**

$$y(t) = 1 - 2e^{-4t} + e^{-8t}$$
