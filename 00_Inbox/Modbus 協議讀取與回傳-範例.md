## 🛠️ 1. 物理層與通訊參數 (Physical Layer)

所有的通訊均透過 CM4 的 `/dev/ttyUSB0` 連接埠進行。

|**參數**|**設定值**|**說明**|
|---|---|---|
|**波特率 (Baud Rate)**|`19200`|傳輸速率|
|**校驗位 (Parity)**|`EVEN` (偶校驗)|PLC 標準校驗方式|
|**資料位 (Data Bits)**|`8`|標準位元組長度|
|**停止位 (Stop Bits)**|`1`|訊號結束位元|
|**站號 (Slave ID)**|**`08`**|CM4 的唯一識別碼|

## 🧠 3. 核心解碼邏輯：位址 0 (40001) 結構

暫存器 0 採用 **數據封裝 (Data Packing)** 技術，將 8 個按鍵與 4 種光源塞進同一個 16-bit 容器中。

### 📌 位元分配圖 (Bit Allocation)

|**Bit 位元**|**功能**|**權重 (Value)**|**解析方式**|
|---|---|---|---|
|**15 ~ 12**|預留|-|保持為 0|
|**11 ~ 4**|**8 顆功能按鍵**|$2^4 \dots 2^{11}$|**Bitmask**: 支援多按鍵同時按下|
|**3 ~ 0**|**光源模式**|$1 \dots 4$|**Index**: 1=R, 2=RE, 3=B, 4=G|

### 📌 計算公式

$$總數值 (Raw) = (\text{按鍵位元總和} \times 16) + \text{光源索引}$$

### 📌 按鍵權重對照表 (Bit 4-11)

|**按鍵**|**#1**|**#2**|**#3**|**#4**|**#5**|**#6**|**#7**|**#8**|
|---|---|---|---|---|---|---|---|---|
|**數值**|**16**|**32**|**64**|**128**|**256**|**512**|**1024**|**2048**|

## 📡 4. 資料封包格式 (Modbus Frame)

### A. 讀取請求與回應 (Function Code 03)

當 PLC 詢問 CM4 的狀態 (Addr 2) 時：

- **PLC 發送**: `08 03 00 02 00 01 [CRC]`
- **CM4 回傳**: `08 03 02 00 01 [CRC]` (回傳數值 1)

### B. 寫入指令 (Function Code 06)

當 PLC 切換按鍵與燈光時：

- **PLC 發送**: `08 06 00 00 [Data_H] [Data_L] [CRC]`
- **CM4 回傳**: `(相同封包內容)` (作為確認 ACK)
