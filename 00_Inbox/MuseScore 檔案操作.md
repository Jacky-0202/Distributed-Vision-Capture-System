## 1. 如何新增檔案 (Create)

新增檔案在 MuseScore 中稱為「建立新分譜 (New Score)」。

- **路徑**：開啟軟體後，在 **「首頁 (Home)」** 頁面點擊 **「新分譜 (New Score)」** 按鈕；或在編輯介面選單點擊 `檔案` > `新建`。
- **流程**：
    
	1. 選擇樂器
	2. 按下 $\rightarrow$  
	3. 按下  Done

## 2. 如何刪除檔案 (Delete)

MuseScore 介面內**無法**真正刪除硬碟裡的檔案。

- **實體刪除 (徹底消失)**：
    
    1. 關閉 MuseScore 程式。
    2. 開啟電腦的「檔案總管 (Windows)」或「Finder (Mac)」。
    3. 進入預設路徑：`文件 > MuseScore4 > Scores`。
    4. 找到該 `.mscz` 檔案，直接按 **刪除 (Delete)**。
        
- **介面清理 (僅移除縮圖)**：
    
    1. 在 MuseScore 首頁的「分譜」清單中。
    2. 點擊檔案縮圖旁的 **「...」**。
    3. 選擇 **「從清單移除 (Remove from list)」**。這不會刪除實體檔案。

## 3. 核心機制說明 (Mechanism)

這是最容易產生誤區的地方，請記住以下邏輯：

> **「MuseScore 是一個編輯器，而不是檔案管理員。」**

- **編輯器屬性**：它的功能是讓你開啟、修改並儲存音樂數據（如同 VS Code 或 Word）。
- **無物理管理權**：它不具備管理電腦硬碟檔案系統（File System）的權限。它顯示的「最近檔案」只是讀取歷史紀錄的 **指標 (Pointer)**。
- **存檔行為**：當你儲存時，它是將當下的狀態封裝成 `.mscz` 壓縮檔寫入硬碟；當你想移動或銷毀檔案時，必須回到作業系統（OS）層級操作。
