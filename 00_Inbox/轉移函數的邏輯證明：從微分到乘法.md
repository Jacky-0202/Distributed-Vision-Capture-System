## 分部積分全解析

轉移函數的成立，建立在一個核心轉化上：**「微分」在 $s$ 領域就是「乘法」**。我們將分兩階段證明。

## 階段一：證明一階微分的轉化式

**目標：** 證明 $\mathcal{L}\{y'(t)\} = sY(s) - y(0)$

1. **根據拉氏轉換定義：**
    
    $$\mathcal{L}\{y'(t)\} = \int_{0}^{\infty} y'(t) e^{-st} dt$$
    
2. **使用分部積分法：**
    
    公式為 $\int u \, dv = uv - \int v \, du$。
    
    - 令 $u = e^{-st}$ $\implies$ $du = -s e^{-st} dt$
    - 令 $dv = y'(t) dt$ $\implies$ $v = y(t)$ (微分的積分即原函數)
        
3. **代入公式展開：**
    
    $$\int_{0}^{\infty} e^{-st} y'(t) dt = \underbrace{\left[ e^{-st} y(t) \right]_{0}^{\infty}}_{\text{Part A}} - \underbrace{\int_{0}^{\infty} y(t) (-s e^{-st}) dt}_{\text{Part B}}$$
    
4. **處理邊界值 (Part A)：**
    
    - 當 $t \to \infty$：$\lim_{t \to \infty} e^{-st} y(t) = 0$ (假設 $s > 0$ 且系統不比指數成長快)。
    - 當 $t = 0$：$e^{0} y(0) = 1 \cdot y(0) = y(0)$。
    - 結果：$(上限 - 下限) = 0 - y(0) = \mathbf{-y(0)}$。
        
5. **處理剩餘積分 (Part B)：**
    
    將常數 $-s$ 提到積分外，負負得正：
    
    $$s \int_{0}^{\infty} y(t) e^{-st} dt = \mathbf{sY(s)}$$
    
    (此積分項即為 $Y(s)$ 的定義)。
    

**結論：** $\mathcal{L}\{y'(t)\} = sY(s) - y(0)$

---
## 階段二：推導至二階微分（遞迴邏輯）

**目標：** 證明 $\mathcal{L}\{y''(t)\} = s^2Y(s) - sy(0) - y'(0)$

我們把 $y''(t)$ 看作是 $y'(t)$ 的一次微分。令 $g(t) = y'(t)$：

1. $\mathcal{L}\{y''(t)\} = \mathcal{L}\{g'(t)\} = sG(s) - g(0)$ (套用階段一的結果)。
2. 將 $G(s) = \mathcal{L}\{y'(t)\} = sY(s) - y(0)$ 代入。
3. 將 $g(0) = y'(0)$ 代入。
    

$$\mathcal{L}\{y''(t)\} = s[sY(s) - y(0)] - y'(0) = \mathbf{s^2Y(s) - sy(0) - y'(0)}$$

---
## 階段三：提煉轉移函數 (Transfer Function)

當我們有一個二階系統微分方程：

$$a y''(t) + b y'(t) + c y(t) = x(t)$$

1. **兩邊施加拉氏轉換：**
    
    $$a \mathcal{L}\{y''(t)\} + b \mathcal{L}\{y'(t)\} + c \mathcal{L}\{y(t)\} = \mathcal{L}\{x(t)\}$$
    
2. **帶入分部積分的結果：**
    
    $$a [s^2Y(s) - sy(0) - y'(0)] + b [sY(s) - y(0)] + c Y(s) = X(s)$$
    
3. **關鍵前提：零初始條件 (Zero Initial Conditions)**
    
    轉移函數定義在系統初始能量為 0 的狀態，故令 $y(0) = 0, y'(0) = 0$。
    
    方程式瞬間簡化為：
    
    $$a s^2 Y(s) + b s Y(s) + c Y(s) = X(s)$$
    
4. **代數提取與移項：**
    
    $$(a s^2 + b s + c) Y(s) = X(s)$$
    
    $$\frac{Y(s)}{X(s)} = \frac{1}{a s^2 + b s + c}$$
    

**最終定義：**

$$H(s) \equiv \frac{\text{輸出}}{\text{輸入}} = \frac{1}{a s^2 + b s + c}$$
