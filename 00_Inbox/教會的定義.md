## 摘要

「教會」不是一棟死的建築物，建築物只能稱之為聖殿、會堂，而是一個與基督有著生命連結的一群人，
是被揀選的，被乎招的。

---
## 教會是「基督的身體」

```
以弗所書 1:22-23
22 又將萬有服在他的腳下，使他為教會作萬有之首。
23 教會是他的身體，是那充滿萬有者所充滿的。
```

很多人以為教會就像志工團體、同鄉會，大家因為有共同的信仰或興趣聚在一起。但這叫「組織」。
以弗所書》給教會下了一個完全不同的定義：「教會是基督的身體」。

組織是可以隨時解散的，但身體是有生命的！耶穌基督是這個身體的「頭」（大腦），而我們是手、腳、眼睛。如果一根手指頭被切斷，
離開了身體，它就缺失了作用，身體的主人也會漸漸變得不方便，同樣的，如果我們離開了教會，就切斷了與耶穌的生命連結，屬靈生命就會漸漸死亡。
教會不是我們「有空才來」的活動中心，它是我們賴以生存的生命共同體。

#### 結論

教會不是我們「有空才來」的活動中心，它是我們賴以生存的生命共同體，。

## 教會是「神的家」

```
提摩太前書 3:15
倘若我耽延日久，你也可以知道在神的家中當怎樣行。這家就是永生神的教會，真理的柱石和根基。
```

保羅說，教會是「神的家」。在外面，我們有各種社會身分（老闆、員工、學生）；但走進這裡，我們就是一家人，因為我們都稱呼同一位神為「阿爸父」。

我們在這個家會知道自己要做什麼，教會哪裡有缺乏就會去幫忙，就像是在家裡，
媽媽煮飯洗衣服，自己就要知道要幫忙洗碗倒垃圾。

但保羅緊接著補了一句極其重要的話：這個家，同時也是「真理的柱石和根基」！
一棟建築物沒有柱子跟地基，地震一來就垮。現在的世界，道德標準天天在變，讓人無所適從。但教會的作用是什麼？教會是真理的柱子！不管外面價值觀怎麼崩塌，教會永遠堅持聖經的標準。

#### 結論

我們來到教會，不只是為了交朋友，更是為了在一個充滿混淆的世代中，找到那條絕對不會改變的真理底線。

---
## 教會是「基督捨己的對象」

```
哥林多前書 6:20
因為你們是重價買來的。所以，要在你們的身子上榮耀神。
```

```
以弗所書 5:25
你們作丈夫的，要愛你們的妻子，正如基督愛教會，為教會捨己。
```

教會到底有多珍貴？
聖經用了一個最極致的比喻來形容神與教會的關係：夫妻。

說：「基督愛教會，為教會捨己」。主耶穌買下教會，不是像大老闆收購一間公司那樣；
祂是像一個深愛妻子的丈夫，為他的新娘付出了生命。

#### 結論

當我們明白這一點，我們還會輕看教會的聚會嗎？我們還會在教會裡隨便批評、
製造分裂嗎？因為我們正在對待的這個團體，是主耶穌用命換來的最愛。
