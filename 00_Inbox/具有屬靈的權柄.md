1. **權柄的賦予：** 定義了教會擁有屬靈上的治理與斷定權。

```
馬太福音 16:19
我要把天國的鑰匙交給你，凡你在地上所捆綁的，在天上也要捆綁；凡你在地上所釋放的，在天上也要釋放。
```

2. **行政與聖靈的結合：** 說明教會的決議是透過人的商討與聖靈的定意共同完成。

```
使徒行傳 15:28
因為聖靈和我們定意不將別的重擔放在你們身上，惟有幾件事是不可少的，
```

```
使徒行傳 16:4-5

```

## 心得

耶穌基督在世上傳道的過程中，第一次談到教會時就賦予教會有綑綁和釋放的屬 靈權柄。
但這捆綁和釋放的權柄須透過一定的行政程序，並在聖靈的見證、定意中才能完成。例如（徒十 五）中的耶路撒冷會議，便是透過會議的行政程序商討，在真理的引導下有聖靈 的見證定意才完成（徒十五 6、16、28）。

- **第 6 節 (商議)：** 建立行政對話的運作。
- **第 16 節 (依據)：** 尋回聖經預言的初衷（重建大衛倒塌的帳幕）。
- **第 28 節 (定意)：** 達成聖靈帶領下的集體共識。

現今的世代是一個環境變動快速、人 心複雜的時代，許多問題在聖經中不一定能有清楚的真理依據，此時就必須透過 提案、按行政程序送到聖職人員會議中討論，取得共識後做成決議（可到教牧資 源網參考本會聖職人員會議決議文之資料）。此為共信之道或共識之道，成為我 們治理教會的遵循原則，為神賦予教會的屬靈權柄。若所有教會能順服、接受會 議制定的條規，則可信心堅固且促進教會增長。
