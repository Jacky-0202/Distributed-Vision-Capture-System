## 🧠 基本概念

- Linux 下刪除檔案常用指令為 `rm`、`rmdir` 與 `find`（搭配 `-delete`）。
- `rm` 直接刪除檔案，不會丟到垃圾桶，也不可復原。
- 建議先確認要刪除的檔案清單，再真正執行刪除指令。
- 若需要可復原的刪除，可使用 `trash-cli` 或 `gio trash`。

---
## 📜 快速對照表

| 需求     | 指令                  | 說明           |
| ------ | ------------------- | ------------ |
| 刪單一檔   | `rm file`           | 基本刪檔         |
| 多檔互動確認 | `rm -i file1 file2` | 每個檔案都會詢問     |
| 整個資料夾  | `rm -r dir`         | 遞迴刪除（危險）     |
| 強制+遞迴  | `rm -rf dir`        | 無提示、忽略錯誤（危險） |
| 只刪空資料夾 | `rmdir dir`         | 資料夾必須為空      |
