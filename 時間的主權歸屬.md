## 摘要

這是一個邏輯命題 (Proposition)的推理過程，要來證明
時間的所有權屬於神，人類僅擁有時間的「使用權」與「管理權」。

---
## 推理證明 (Deductive Proof)

#### 從「創造秩序與規則」推理 (Creation Order Argument)

- 時間是神所設立的「受造界秩序」與「規則」。

```
創世記 1:14
神說：「天上要有光體，可以分晝夜，作記號，定節令、日子、年歲，
```

所以時間不是自然存在的，而是神**特意創造**的度量衡

- 神是「自有永有」的，祂位於時間的秩序之外。

```
出埃及記 3:14
神對摩西說：「我是自有永有的」；又說：「你要對以色列人這樣說：『那自有的打發我到你們這裡來。』」
```

```
彼得後書 3:8
親愛的弟兄啊，有一件事你們不可忘記，就是主看一日如千年，千年如一日。
```

祂是昔在、今在、以後永在的全能者，不受時間流逝的限制，它可以再過去、
可以在未來、也在現在。

- 結論

	- 既然神是「時間規則的制定者」**（創造了日夜節令），且祂自己**「不受時間限制」**（自有永有），那麼祂對時間擁有絕對的**「所有權」**與**「支配權」。

	- 人類作為受造物，活在神所規劃的「時間規則」內，因此我們對時間只有「使用權」，必須按照制定者的規則（神的心意）來運行。
